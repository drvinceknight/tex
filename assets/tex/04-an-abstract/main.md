### An abstract

(Keywords: abstract.)
[Video demo](https://www.youtube.com/watch?v=AQP1lM7jWGY)

The following will add an abstract to your document:

```language-latex
\begin{abstract}
    This document contains some basic LaTeX code that will be useful to me
    in the future.
\end{abstract}
```
