### Arithmetic operators

Arithmetic operators are available in \\(\LaTeX\\):


```language-latex
\begin{itemize}
	\item \(a+b\)
	\item \(a-b\)
	\item \(-a\)
	\item \(ab\)
	\item \(a\cdot b\)
	\item \(a\times b\)
	\item \(a/b\)
	\item \({a\over b}\)
	\item \(\frac{a}{b}\)
\end{itemize}
```
